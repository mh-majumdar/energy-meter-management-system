Database Model Structure

User Model (UserMixin)

id: Integer (Primary Key)
username: String(150) - Unique
email: String(150) - Unique
password_hash: String(256)
date_created: DateTime with timezone
Relationships:

• One-to-One with Profile (backref: 'user')

• One-to-One with MeterAccount (backref: 'meter_account')

Profile Model

id: Integer (Primary Key)
user_id: Integer (Foreign Key)
fathers_name: String(150)
mothers_name: String(150)
date_of_birth: String(20)
phone: String(20)
guardians_name: String(150)
guardians_phone: String(20)
hall_for_residence: String(150)
hall_for_roll: String(150)
blood_group: String(10)
religion: String(50)
nationality: String(50)
nid_number: String(20)
gender: String(20)
marital_status: String(20)
academic_level: String(50)
faculty: String(150)
department: String(150)
session: String(20)
year_term: String(30)
student_id: String(20)
meter_id: String(100)
connection_type: String(50)
phone_no: String(100)
installation_date: String(20)
supply_voltage: String(50)
updated_at: DateTime (auto-update)

Address Fields: Present & Permanent addresses with division, district, upazilla, post_office, village, house_name, house_no, road_no

MeterAccount Model

id: Integer (Primary Key)
meter_id: String(20) - Unique
user_id: Integer (Foreign Key)
balance: Float (default: 90.0)
created_at: DateTime
updated_at: DateTime (auto-update)
Relationships:

• One-to-One with User (backref: 'meter_account')

• One-to-Many with Transaction (backref: 'account')

• One-to-Many sent_transactions (backref: 'sender')

• One-to-Many received_transactions (backref: 'receiver')

Transaction Model

id: Integer (Primary Key)
meter_account_id: Integer (Foreign Key)
amount: Float
transaction_type: String(20) ['add', 'send', 'receive']
description: String(255)
reference_id: String(100)
created_at: DateTime
sender_id: Integer (Foreign Key - nullable)
receiver_id: Integer (Foreign Key - nullable)
